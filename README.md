# Aayush Ki Mehnat 📚

**Aayush Ki Mehnat** is the ultimate digital hub for RTU (Rajasthan Technical University) B.Tech students. This platform provides a comprehensive collection of free handwritten notes, textbooks, previous year question papers, and assignments for all semesters and branches.

## 🚀 Features

- **Handwritten Notes**: High-quality, easy-to-understand notes from top students.
- **Digital Library**: Access a wide range of engineering textbooks in PDF format.
- **Previous Year Papers**: Practice with actual RTU exam papers from past years.
- **Assignments & Lab Manuals**: Get help with your practical work and semester assignments.
- **Branch-wise Organization**: Easily navigate through materials for CS, IT, ECE, ME, CE, and more.

## 🛠️ Tech Stack

- **Frontend**: React.js with Vite
- **Styling**: Tailwind CSS
- **Backend/Database**: Firebase (Firestore & Authentication)
- **Deployment**: Vercel
- **Analytics**: Google Analytics (gtag.js)

## 📦 Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/aayushsh2003/aayushkimehnat.git
   cd aayushkimehnat
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file in the root directory and add your Firebase configuration (see `.env.example`).

4. **Start the development server**:
   ```bash
   npm run dev
   ```
